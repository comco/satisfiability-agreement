% Hardness of the monadic logic with equivalenceIn this section we show that the (finite) satisfiability of monadic first-orderlogic with a single equivalence symbol $\Lvp\Fo\nonv1\Eea1\noag$ is$\ceNExpTime2$-hard by reducing the doubly exponential tiling problem to suchsatisfiability. Our strategy is to employ a counter setup of $\szu$ unarypredicate symbols to encode the exponentially many positions of a binaryencoding of a doubly exponentially bounded quantity, encoding the coordinaes ofa cell of the doubly exponential tiling square.Consider the counter setup $\CS(\szu) = \seq{\suu1,\suu2,\dots,\suu\szu}$ for$\szu \in \pNats$. Recall that the intention of a counter setup is to encode anarbitrary exponentially bounded value at every element of a structure.Let $\DS(\szu, \sco) = \CS(\szu) + \seq{\sco, \sd}$ be a predicate signatureenriching $\CS(\szu)$ with the unary predicate \emph{control symbol} $\sco$and the builtin equivalence symbol $\sd$.\begin{definition}Let $\StrA$ be a $\DS = \DS(\szu, \sco)$-structure. Define the function$\gls{Data-D-A} : \pset\domA \to \tBitstrings{2^\szu}$ by:\[  \Data\DS\StrA_\posp \setS = \begin{cases}  1 &\text{if } \data\CS\StrA\ea = \posp \text{ and } \StrA \vDash \sco(\ea)  \text{ for some } \ea \in \setS \\  0 &\text{otherwise.}  \end{cases}\]\end{definition}Let $\StrA$ be a $\DS = \DS(\szu, \sco)$-structure and let $\relD =\at\StrA\sd$.\begin{definition}Define the quantifier-rank-$1$ $\vFoF2\DS$-formula $\gls{fZero-D-x}$ by:\[  \fZero\DS(\xx) = \forall\yy \sd(\yy,\xx) \limp \lnot \sco(\yy).\] \end{definition}Then $\StrA \vDash \fZero\DS(\ea)$ iff $\Data\DS\StrA(\ea) = \benc{0}$.TODO:\begin{definition}\[  \feqpos\DS(\xx,\yy) = \sco(\xx) \land \sco(\yy) \land \feq\CS(\xx,\yy).\]\end{definition}\begin{definition}\[  \fSucc\DS(\xx,\yy) = \exists\zz\exists.\]\end{definition}
